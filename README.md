SFML2 - Random Games (WIP)
=================

A little collection of simple games made with SFML2. Some of them can be unfinished !

To compile it, you'll need:

- GCC >= 4.7.0
- [SFML](http://www.sfml-dev.org/) >= v2.1 (2013-07-25 fork)
